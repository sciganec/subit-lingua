# SUBIT‑Lingua Extended Lexicon (512 Words)

This file defines the **512‑word extended lexicon** of SUBIT‑Lingua.  
It expands the Core Lexicon (204 words) with:

- 128 systematic derivatives  
- 96 cross‑axis composites  
- 64 processual chains  
- 20 meta‑concepts  

All entries remain structurally interpretable and map directly to SUBIT‑RFC.

# 2. Systematic Derivatives (128 Words)

These are **axis‑preserving expansions** created by combining:

```
[Axis Root] + [Modifier Root]
```

Each axis contributes **32 derivatives**:

- 8 WHO‑axis  
- 8 WHERE‑axis  
- 8 WHEN‑axis  
- × 4 vowel states  
= 32 per axis  
= **96 total**

Plus 32 mixed‑axis expansions.
