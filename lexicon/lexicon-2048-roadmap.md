# SUBIT‑Lingua Lexicon Roadmap (2048 Words)

This document defines the roadmap for expanding the SUBIT‑Lingua lexicon from the current **512‑word Extended Lexicon** to a fully structured **2048‑word lexicon**.

The expansion preserves:

- full SUBIT isomorphism  
- strict morphological regularity  
- transparent semantics  
- axis‑based compositionality  
- machine‑readability  

The 2048‑word lexicon is not a random enlargement.  
It is a **systematic, layered, mathematically clean expansion** of the SUBIT‑Lingua Core.

---

# 1. Target Structure: 2048 Words

The 2048‑word lexicon is composed of:

| Layer | Count | Description |
|-------|-------|-------------|
| Core Lexicon | 204 | Base + doubled + structured + thematic |
| Extended Lexicon | 512 | Systematic derivatives + composites + chains |
| **New Layer A** | **512** | Axis‑balanced semantic expansions |
| **New Layer B** | **512** | Higher‑order structured compounds |
| **New Layer C** | **320** | Domain‑specific semantic fields |
| **New Layer D** | **256** | Multi‑axis conceptual matrices |
| **New Layer E** | **244** | Reserved for future SUBIT‑Lingua modules |
| **Total** | **2048** | — |

The roadmap below defines how each new layer is constructed.

# 2. Layer A — Axis‑Balanced Semantic Expansions (512 Words)

This layer extends the lexicon by generating **systematic, axis‑balanced expansions** of existing forms.

### 2.1 Method

For each base root (12 total):

```
CV × (all 4 vowels) × (all 3 axes) × (3 semantic modes)
```

Semantic modes:

1. **State** — static condition  
2. **Process** — dynamic transformation  
3. **Relation** — interaction between axes  

This yields:

```
12 roots × 4 vowels × 3 axes × 3 modes = 432 words
```

The remaining 80 words fill structural gaps (e.g., missing WHO–WHERE–WHEN triads).

### 2.2 Purpose

- enriches the lexicon without breaking regularity  
- expands semantic granularity  
- preserves axis symmetry  

---

# 3. Layer B — Higher‑Order Structured Compounds (512 Words)

These are **fully regular compounds** of the form:

```
[Structured Form] + [Modifier]
```

Where:

- structured form = CV‑CV‑CV (64 total)
- modifier = root or doubled form (12 + 48 = 60)

We select:

```
64 structured × 8 canonical modifiers = 512
```

Modifiers are chosen to maximize:

- semantic clarity  
- axis diversity  
- morphological stability  

### 3.1 Example Patterns

```
KA‑TE‑MU + KA = active subject → self‑directed action
TA‑KO‑ME + ME = deep spatial root → temporal shaping
MA‑KU‑MA + TU = cycle‑energy → intensity modulation
```

# 5. Layer D — Multi‑Axis Conceptual Matrices (256 Words)

This layer introduces **matrix‑based combinations**:

```
[WHO] × [WHERE] × [WHEN] × [State]
```

Where **State** is one of:

- A (00)  
- E (01)  
- O (10)  
- U (11)  

This yields:

```
4 × 4 × 4 × 4 = 256
```

Each matrix entry is a **canonical conceptual unit**, representing:

- a subject in a spatial configuration at a temporal phase  
- a structural relation across axes  
- a stable semantic coordinate  

These 256 words form the **semantic backbone** of the 2048‑word lexicon.

It ensures the lexicon can grow **without breaking the 2048 structure**.

# 7. Generation Principles

All new words must follow:

### 7.1 Structural Rules

- CV, CV‑CV, or CV‑CV‑CV forms only  
- no consonant clusters  
- no codas  
- no diphthongs  
- no irregular affixes  

### 7.2 Semantic Rules

- axis‑aligned  
- compositional  
- transparent  
- reversible  
- machine‑interpretable  

### 7.3 Distribution Rules

- equal representation of WHO, WHERE, WHEN  
- equal representation of all four vowels  
- balanced semantic fields  
- no overloaded roots
