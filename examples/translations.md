# SUBIT‑Lingua — Translations (SUBIT ↔ English)

This document provides **parallel translations** between SUBIT‑Lingua and English.  
Each example includes:

- the SUBIT‑Lingua expression  
- a literal structural gloss  
- a natural English interpretation  

All examples follow the minimal syntax rules of SUBIT‑Lingua.

---
