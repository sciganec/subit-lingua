# SUBIT‑Lingua Playground — Interactive Scenarios

This module introduces interactive patterns for working with SUBIT‑Lingua in
dialogue with an AI agent. Each scenario provides a structural prompt, an
expected type of agent response, and a pattern for continuing the interaction.

The goal is not translation.  
The goal is structural communication.

---

# 1. Atomic Interaction

### Scenario 1.1 — State Inquiry
**You → Agent:**  
`KA?`

**Agent responds with:**  
- the structural meaning of KA  
- its axis and bigram  
- a minimal interpretation

**Goal:**  
Learn how the agent interprets atomic states.

---

### Scenario 1.2 — Axis Comparison
**You → Agent:**  
`KA TA MA — compare.`

**Agent responds with:**  
- differences across axes  
- how the same bigram behaves in WHO/WHERE/WHEN

**Goal:**  
Observe axis‑based contrast.
